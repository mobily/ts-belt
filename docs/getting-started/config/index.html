TS and ESLint config

TypeScript​

Update strictNullChecks and strict in your tsconfig.json (optional but recommended):

{
  "compilerOptions": {
    …
    "strict": true,
    "strictNullChecks": true
  }
}

ESLint​

You may also want to add eslint-plugin-functional to your ESLint config. TS Belt returns immutable data, so enabling the functional/prefer-readonly-type rule is highly recommended.

{
  "plugins": ["functional"],
  "rules": {
    "functional/prefer-readonly-type": "error"
  }
}
