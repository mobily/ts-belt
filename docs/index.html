Getting started

üîß TS Belt is a library for functional programming in TypeScript. It solves the problem of the existence of both undefined and null. TS Belt is inspired by the Belt module for ReScript/Reason.

Features#

  • lightweight, no dependencies
  • type safety, full TypeScript support
  • high tests coverage
  • all functions are curried
  • get rid of undefined and null values in your project in a convenient, functional way (use either Option or Result)
